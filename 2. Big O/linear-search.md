# 🔍 Linear Search in JavaScript (O(n))

## 📌 Definition

Linear Search ek simple algorithm hai jisme hum array ke har element ko ek ek karke check karte hain jab tak desired item mil na jaye.

<!-- ======= interviews answer ======= -->

Linear search me hum array ke elements ko sequentially ek ek karke check karte hain, isliye iska time complexity O(n) hota hai.
Even agar multiple loops ho, hum constants ignore kar dete hain, so overall complexity still O(n) hi rehti hai.

Linear search iterates through each element one by one, giving a time complexity of O(n).
Even if there are multiple independent loops, we drop constant factors, so the overall complexity still remains O(n).
